# Asteroids Game 
Created by Nathan Wong \
Inspired by the original Asteroids Game by Atari

## Features
* Built using the openFrameworks C++ toolkit
* Four difficulties
* Game difficulty increases as time goes on
* Collisions between Ship and Asteroids (along with Asteroid on Asteroid collisions)
* Ship Projectile and Asteroid Collisions
* Multiple Asteroid sizes (destroying a big asteroid spawns smaller asteroids)
* Cool particle effects 
* Ship movement and steering simulate real physics (thrusters, momentum)
* and much more
